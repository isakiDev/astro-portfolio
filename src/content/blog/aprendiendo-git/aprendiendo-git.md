## ¿Qué es Git?

Git fue creado por **Linus Torvalds** y se utiliza principalmente para:

- Controlar versiones de código
- Trabajar en equipo sin sobrescribir cambios
- Volver a estados anteriores del proyecto
- Mantener un historial claro de modificaciones
